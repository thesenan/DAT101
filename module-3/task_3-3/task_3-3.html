1. What is the difference between invoking and creating a function?

    2. Creating a function defines its behavior, while calling a function executes that behavior.

  2. Explain the syntax regarding semicolons and code blocks.

    3. Semicolons are used to terminate statements, and code blocks are enclosed in curly braces: {}.

  3. How do we separate parameters from each other when creating a function?

    4. Parameters are separated by commas when creating a function.

  4. Explain how parameter names are related during the invocation and creation of a function.

    5. The parameter name in the function are just placeholders. When a function is called, the actual values passed in are assigned to these parameters.

  5. Explain why a parameter is undefined when a function call is missing a parameter.

    6. A parameter becomes undefined if no value is passed for it during the function call.

  6. Explain what happens when we use recursive functions.

    7. A recursive function keeps calling itself until it reaches a stop condition.
